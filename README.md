# Vile's Keybind Manager

Vile's Keybind Manager is a browser-based keybinding layout planner for games and applications. It lets you visually design and compare full keyboard + mouse setups, save multiple profiles, and export/import them as JSON so you never lose a layout.

## Features

- Interactive keyboard + mouse grid for assigning actions or labels  
- Multiple named profiles (e.g., different characters, specs, or games)  
- Clone, rename, and delete layouts with a few clicks  
- Export and import profiles as JSON for easy backup and sharing  
- Automatic saving to browser `localStorage` (no login or backend required)  
- Clean dark UI built with pure HTML, CSS, and vanilla JavaScript  

---

## Getting Started

### Use in the browser

1. Open the live demo:  
   https://super-mf-villain.github.io/Vile-s-Keybind-Manager/
2. Click any key or mouse slot to assign an action/ability.
3. Use the profile controls to create, switch, and manage layouts.
4. Use **Export** to download your layouts as JSON and **Import** to load them later.

### Run locally

If you prefer to run it from your machine:

1. Download or clone this repository.
2. Open `index.html` in any modern web browser.
3. All functionality is available offline; data still saves to your browserâ€™s `localStorage`.
